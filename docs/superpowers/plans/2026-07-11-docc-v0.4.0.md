````markdown
# 事后文档收口(postflight)

## 步骤

1. **取核对清单(报告驱动)**:`node "${CLAUDE_PLUGIN_ROOT}/scripts/doc-preflight.mjs" --cwd .`——以 `driftCandidates` + `docDrift` 为本轮核对清单(口径与盖章范围一致)。`mapIssues` 非空先修映射(元素为 `{code, message}`,按 code 分流;`baseline-corrupt` 由第 7 步盖章自然修复)。
````

```js
// git 子进程输出上限:大仓清单/批量哈希超过 Node 默认 1MB 会被 ENOBUFS 杀死
export const GIT_MAX_BUFFER = 64 * 1024 * 1024;

/**
 * 批量哈希:单次 git 进程处理全部路径(逐文件 spawn 在千文件仓库慢约 250 倍)。
 * 整批失败(任一路径不可读/行数不符)返回 null——调用方回退逐文件,保 unhashable 告警粒度。
 */
export function gitHashObjects(cwd, relPaths) {
  if (!relPaths.length) return {};
  try {
    const r = spawnSync("git", ["hash-object", "--no-filters", "--stdin-paths"], {
      cwd,
      encoding: "utf-8",
      input: `${relPaths.join("\n")}\n`,
      maxBuffer: GIT_MAX_BUFFER,
    });
    if (r.status !== 0) return null;
    const lines = r.stdout.split(/\r?\n/).filter(Boolean);
    if (lines.length !== relPaths.length) return null;
    const out = {};
    relPaths.forEach((p, i) => {
      out[p] = lines[i];
    });
    return out;
  } catch {
    return null;
  }
}
```
